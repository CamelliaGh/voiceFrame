# Fonts Directory

This directory contains custom fonts for the Audio Poster templates.

## Required Fonts

To match the photo layout exactly, you'll need these font files:

1. **script.ttf** - A handwritten/script style font for elegant text like "Hi honey it's Daddy, I love you"
2. **elegant.ttf** - An elegant serif font for formal designs
3. **modern.ttf** - A modern sans-serif font for contemporary designs

## Font Sources

You can download free script fonts from:
- Google Fonts (https://fonts.google.com)
- Font Squirrel (https://www.fontsquirrel.com)
- DaFont (https://www.dafont.com)

## Recommended Script Fonts

For the script style matching the photo:
- "Dancing Script" (Google Fonts)
- "Great Vibes" (Google Fonts) 
- "Allura" (Google Fonts)
- "Alex Brush" (Google Fonts)

## Installation

1. Download the font files
2. Place them in this directory with the exact names: script.ttf, elegant.ttf, modern.ttf
3. The system will automatically use these fonts when generating PDFs

## Fallback

If custom fonts are not available, the system will fall back to system fonts or default fonts.
